# RNG

RNG is a fresh Layer-1 chain launch focused on deterministic consensus and fair,
verifiable randomness for gaming and autonomous systems.

## v1.0 Genesis

- Chain name: `RNG`
- Smallest unit: `roshi`
- Genesis message: `Life is a random number generator`
- Network: new genesis start (clean history)

## Mission

Build a resilient random-number economy with transparent issuance, auditable state,
and simple operator workflows.

## Status

`v1.0` marks the beginning of the RNG chain from genesis.
